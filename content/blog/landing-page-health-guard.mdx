Get notified when something is broken on your webpage.

## But please remember, Google Ads Script has its constraints...

There can be outages in the script. Google Ads Scripts does not allow you to monitor 24/7, just on an hourly basis. Also there are limits in using this script (I haven't experienced this yet with 40 accounts in MCC but you might if you have more).

Therefore (and also due to the enormous demand of my Google Ads script on LinkedIn), I have decided to start developing a tool which secures the links of your Google Ads, but also Facebook Ads, TikTok Ads & more.

## What the tool monitors

The tool will 24/7 monitor your:
- Load time
- SSL certificate validity
- HTTP status codes
- Mobile-friendliness
- Redirect chains
- Presence of Google Tag Manager
- Google Analytics implementation
- Google Ads conversion tracking
- Conversion API set-up
- And way more to come

## Why create a separate tool?

Because not everyone knows how to run Google Ads Scripts. Because Google Ads Scripts has its limits. Because Google Ads Scripts can not run 24/7. Because there is not anything similar available for Facebook, TikTok Ads.
